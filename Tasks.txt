*git add - многофункциональная команда, она используется для добавления под версионный контроль новых файлов, для индексации изменений, а также для других целей
git init-Эта команда создает пустой репозиторий Git - в основном каталог .git с подкаталогами для объектов, refs / headers, refs / tags и файлами шаблонов
git commit-Он сохраняет текущее содержимое индекса в новом коммите вместе с лог-сообщением от пользователя, описывающим изменения
git push- означает мерж локальных изменений в удаленный репозиторий
